# =-= MARKETOON =-=
## MANUAL DO USUÁRIO
O _Marketoon_ é um sistema de controle de vendas que tem como objetivo auxiliar um artista, que queira anunciar a venda do seu produto, e intermediar a compra do usuário com o vendedor.
